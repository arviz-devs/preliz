Dependencies
============
PreliZ is tested on Python 3.8+.

Installation
============

For the latest release you can do:

.. code-block:: bash

  pip install preliz

The latest development version can be installed from the main branch using pip:

.. code-block:: bash

  pip install git+https://github.com/arviz-devs/preliz.git
